# Changelog

All notable changes to AutoFlow workflows will be documented in this file.

## [1.0.0] — 2026-03-30

### Added
- **FlowScribe Lite** — Content repurposing: 1 blog post → 4 platforms (Twitter, LinkedIn, Instagram, Facebook)
- **LeadPilot Lite** — AI cold email writer with personalized outreach
- **SupportFlow Lite** — Simple AI chatbot using company context
- **InboxZero Lite** — Gmail classifier (urgent/important/info/spam)
- **SocialPulse Lite** — Reddit trend monitor with AI content suggestions
- **ClientFlow Lite** — Automated client onboarding (form → email → log)
- **DataForge Lite** — URL data extractor with AI-powered structuring
- **VoiceAgent Lite** — Call logger for Vapi.ai and Bland.ai webhooks
- Contributing guide and issue templates
- MIT License
- GitHub Actions workflow validation
